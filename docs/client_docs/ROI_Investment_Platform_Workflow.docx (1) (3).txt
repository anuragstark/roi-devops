ROI Investment Platform - Work Flow Document 
1. Overview 
This document describes the workflow, structure, and functionality of the ROI-based investment 
platform. The platform includes ROI plans, referral income, salary income, multi-level structure, 
breakdown management, and integrated deposit/withdrawal systems using USDT and INR. It 
supports admin, staff, distributor, and white-label systems (B2B/B2C).

3. WALLET MANAGEMENT 
Dual-wallet system for INR and USDT balances. 
3.1 INR Wallet 
● Deposit via: 
 
○ UPI / NEFT / IMPS (manual or API) 
 
● Upload payment proof (if manual). 
 
● View transaction status: Pending / Approved / Rejected. 
 
3.2 USDT Wallet 
● Deposit using blockchain address (TRC20/ERC20). 
 
● View TX ID and confirmation status. 
 
● Auto-credit on confirmation. 
 
3.3 Wallet Functions 
● Withdraw: Choose wallet type, enter amount, confirm via OTP/2FA. 
 
● View History: Transaction type, amount, TX ID, status, date. 
 
● Currency Converter: INR ⇄ USDT (live API rates). 
 
Flow: 
 Wallet → Deposit / Withdraw → Confirmation → Status Tracking 
 
4. INVESTMENT PLANS 
User can invest directly from wallet. 
4.1 Browse & Buy Plans 
● View all available plans with: 
 
○ ROI %, Duration, Minimum Investment, ROI Frequency (daily/weekly/monthly). 
 
● Select plan → Confirm purchase using Wallet balance. 
 
● Plan activates instantly after payment confirmation. 
 
4.2 Active Plans 
● Displays ongoing investments: 
 
○ Start Date, Maturity Date, ROI %, Earned ROI, Status. 
 
● ROI credited automatically as per schedule. 
 
4.3 Plan History 
● Shows completed/expired plans. 
 
● Option to reinvest or upgrade to higher plan. 
 
Flow: 
 Dashboard → Choose Plan → Pay via Wallet → Active Plan → Auto ROI 
Credit 
 
5. ROI & INCOME 
Tracks all income types generated from investments and referrals. 
5.1 ROI Earnings 
● Auto-credited per plan schedule. 
 
● Display ROI source, plan ID, credit date, and amount. 
 
● ROI credited to ROI Wallet. 
 
5.2 Salary Income (Qualification) 
● If user qualifies (based on referrals): 
 
○ 2 Paid Users within 72 hrs, OR 
 
○ 10 Free Users within 72 hrs. 
 
● Once qualified → Monthly Salary Income auto-credited. 
 
5.3 ROI Boost 
● When 50+ active referrals (Free/Paid combined) → ROI % boost applied. 
 
● Boost percentage defined by admin, visible to user on dashboard. 
 
Flow: 
 Investment → Qualification → Auto ROI & Salary Credit → Boost 
Updates 
 
6. REFERRAL SYSTEM 
Multi-level referral structure for network growth. 
6.1 Invite & Share 
● Generate referral link or QR code. 
 
● Share via WhatsApp, Telegram, Email, or Copy Link. 
 
6.2 Referral Tree 
● View team hierarchy (level-wise): 
 
○ Level 1 → Direct Referrals 
 
○ Level 2 → Referrals of Directs 
 
● Displays total members, paid/free users, qualification stats. 
 
6.3 Referral Income 
● Earnings from direct and indirect referrals. 
 
● Breakdown: Level, Member Name, Status, Reward Amount. 
 
Flow: 
 Referral → Invite → Team Tree → Income Report 
 
7. BREAKDOWN WALLET SYSTEM 
When investment made: 
● 20% held for system operations. 
 
● 80% credited to Breakdown Wallet for ROI and refunds. 
 
● Refund eligibility shown with timeline. 
 
Example: 
 Investment 100 USDT → ROI 20 USDT → Debit 10 USDT → Refund 70 USDT after period. 
Flow: 
 Invest → 80% Breakdown Credit → ROI Credit → Refund Timeline
